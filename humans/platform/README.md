# Platform Distribution

Operators for marketplace and integration distribution: Shopify App Store, Stripe Marketplace, Slack/HubSpot/Salesforce app directories, Chrome Web Store, VS Code marketplace, Zapier integrations.

## What we look for

- Specific marketplace experience (which marketplace, which apps)
- Understanding that each marketplace has its own ranking and discovery rules
- Comfort with the technical side (manifests, OAuth, review processes)

## Anti-fits

- Generic "go-to-market" advisors with no marketplace specifics
- Anyone who treats every marketplace as the same
