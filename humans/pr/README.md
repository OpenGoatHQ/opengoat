# PR & Influencer

Operators for press, podcasts, and creator partnerships: founder PR, journalist outreach, HARO/Connectively/Featured, podcast tours, influencer/creator deals.

## What we look for

- Verifiable placements (named publications, named podcasts)
- Existing relationships, not cold-only reach
- Honest about what's earnable vs paid

## Anti-fits

- PR firms that won't disclose past clients
- Influencer agencies that just blast lists
