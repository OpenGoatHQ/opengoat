Operators who run search engine and AI-search visibility: technical SEO, content SEO, topical authority, programmatic SEO, link building, and Generative Engine Optimization (GEO) for ChatGPT/Perplexity/Claude.

## What we look for

- Verifiable traffic gains on named projects (anonymized OK with NDA)
- Comfort with both technical (Core Web Vitals, schema, crawl) and content sides
- Honest about what only works at certain scales
- Aware that 2024+ Google updates changed the game

## Anti-fits

We don't list operators who:
- Promise specific rankings
- Sell PBN/private link networks
- Recommend mass AI-generated content
